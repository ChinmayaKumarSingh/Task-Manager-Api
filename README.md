# Task Manager API

A Spring Boot REST API for managing tasks.

## Features
- Create Task
- Get All Tasks
- Delete Task
- Layered Architecture

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Endpoints

GET /tasks  
POST /tasks  
DELETE /tasks/{id}
